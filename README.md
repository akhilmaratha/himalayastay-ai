# HimalayaStay AI

AI-assisted direct booking and guest experience management platform for rural homestays.


## Description

A platform helping homestay owners manage direct bookings, rooms, availability, and guest reviews using AI-powered insights.


## Tech Stack

Frontend:
- Next.js
- React
- Tailwind CSS

Backend:
- Node.js
- Next.js API Routes

Database:
- MongoDB

AI:
- Gemini API

Authentication:
- NextAuth.js
